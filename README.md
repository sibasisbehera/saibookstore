# SAI Book Store — GST Bill Generator

A single-file, responsive web app for generating GST-compliant bills for **SAI Book Store** (Khelar, Dist-Puri). Runs entirely in the browser — no server, no database, no monthly cost. Hostable free on GitHub Pages or Netlify.

## Core Features

### 1. Login + Welcome
- Creative split-screen login (animated floating book/printer/pen icons, pulsing Sai Baba logo).
- On success -> animated welcome screen greeting **Debi Prasad Mohanty**, then **Enter App**.
- "Sign out" button in the top nav.
- **Stay logged in on refresh:** login state is kept in `sessionStorage`, so a page reload does not force re-login. Closing the tab logs you out.

### 2. Bill Editor (the "Bill" tab)
- **Invoice No.** — auto-generated, unique every time, format `DDMMYY-HHMMSS` (e.g. `060626-141222`). Editable.
- **Date** — auto-set to today, editable.
- Customer name, V-No/phone, address (optional), bill type (Cash/Credit).
- **GST rate** dropdown — 0%, 5%, 12%, 18% (default), 28%. Splits into SGST + CGST automatically.
- **Items table** — add/delete rows; auto-calculates amount per row.
- **Discount** — toggle between flat `Rs.` and `%` (defaults to `%`).
- **Live summary** — Taxable value, SGST, CGST, Sub Total, Discount, Total + amount in words (Indian Lakh/Crore format).

### 3. Output options
- **Print** — opens browser print dialog with clean A4 layout.
- **Preview** — full A4 preview in a modal (mobile-friendly, scrolls and fits screen).
- **Download PDF** — file named `saibookstore-<invoiceNo>.pdf`.
- **WhatsApp** — opens WhatsApp with customer number + bill summary pre-filled, and downloads the PDF for manual attach. Reuses a single named tab so it doesn't stack new tabs.
- **Save** — saves the bill to in-session History.

### 4. Edit Previous Bill (PDF round-trip)
- Every generated PDF has the full bill data **embedded invisibly** inside it (between `SAI_BILL_DATA_START` ... `SAI_BILL_DATA_END`).
- Click **Edit Previous Bill** -> upload a PDF made by this app -> the form auto-fills -> edit -> re-download.
- **Limitation:** only works on PDFs generated by *this* app. Old PDFs or PDFs from other tools cannot be read back.

### 5. History (the "History" tab)
- Lists bills saved via the **Save** button during the current session.
- Per bill: **Load & Edit** (reopens in editor), **Preview**, **PDF**, **Delete**.
- **Session only** — history clears when the tab is closed (by design; no permanent bill storage).

### 6. Settings (the "Settings" tab)
- **Shop Information** — name, address, mobile, GSTIN, default GST rate, + **Logo show/hide** toggle.
- **Bank Details** — account holder (highlighted), account no, bank, branch, IFSC.
- **Terms & Signature** — terms text + authorised-by name.
- Each card has an **Edit** button (view -> edit -> save).
- **Settings persist permanently** in `localStorage` — change once, every future bill uses the new values, even after closing the browser.

## Storage Model (important)

| Data | Where | Lifetime |
|------|-------|----------|
| Shop settings (name, GST, bank, terms, logo toggle) | localStorage | Permanent — survives reload + tab close |
| Login state | sessionStorage | Until tab is closed |
| Draft bill (auto-saved as you type) | sessionStorage | Until tab closed; restorable after reload |
| Bill history | in-memory (session) | Cleared on tab close |

**Draft recovery:** if the page reloads while you're typing, a yellow banner offers to **Restore** the unsaved draft. Cleared automatically when you Save or start a New Bill.

> There is intentionally **no permanent storage of bills**. The PDF files you download (and optionally back up to Google Drive) are the system of record. Deliberate choice to keep the app simple and private.

## Deployment (free)

### GitHub Pages
1. Create a GitHub account -> New repository named `saibookstore` (Public).
2. Upload `index.html` and `.nojekyll`.
3. Settings -> Pages -> Source: Deploy from branch -> `main` -> `/ (root)` -> Save.
4. Live at `https://<username>.github.io/saibookstore`.

### Netlify
1. Sign up at netlify.com.
2. Drag-and-drop `index.html` onto the dashboard.
3. Instant live URL.

### Local
Just open `index.html` in any browser — works offline (except WhatsApp link and CDN scripts, which need internet on first load).

## Known Limitations / Future Ideas
- **Login security** is front-end only (credentials visible in source). Fine for a private single-shop tool; not for public/multi-user use.
- **No cross-device sync** — settings live in the browser they were entered in. Google Drive integration discussed as a future add-on.
- **PDF re-edit** works only for PDFs this app generated.
- **Excel export removed** — layout was distorted; PDF round-trip replaced its "record" purpose.
- Future ideas: Google Drive save, multi-rate GST per bill, barcode scanning, monthly GST summary report.
